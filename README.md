# client

## Project setup
```
npm install
```

### Для лайв-тестов
```
npm run serve
```

### Перед билдом и запуском сервера проверить директории в Express сервере!

### Для тестирования "как на продакшн" через Express
```
npm run build
nodemon testServer.js
```

### Lints and fixes files
```
npm run lint
```
