# client

## Установка
```
npm install
```

### Для лайв-тестов
```
npm run serve
```

### Перед билдом и запуском сервера проверить директории в Express сервере!

### Для тестирования "как на продакшн" через Express
```
npm run build
nodemon server/app.js
```

### Если хотите, чтоб на вас наорали и унизили
```
npm run lint
```
